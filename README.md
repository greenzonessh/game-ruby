# README

## Requirement
* Install MySQL / MariaDB on port 3306 and setup your configuration on /config/database.yml
```
  username: root
  password:
```

## Installation
```
  rake db:create
  rake db:migrate
```

## Run
```
  rails s
```
 Open browser http://localhost:3000 <br>
 You can add Vocabulary http://localhost:3000/vocabularies
